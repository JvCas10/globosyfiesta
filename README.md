# 🎈 Globos y Fiesta - Sistema de Gestión

Sistema completo de gestión para negocio de globos y decoraciones.

## 🚀 Tecnologías

**Backend:**
- Node.js + Express
- MongoDB Atlas
- Cloudinary (imágenes)
- JWT (autenticación)

**Frontend:**
- React 18
- React Router
- Axios

## 📦 Instalación

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Configurar variables en .env
npm run dev
```
### Frontend
```bash
cd frontend
npm install
npm start
